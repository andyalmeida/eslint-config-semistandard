# ESLint Config

## Whats included?

- Semi-Standard config base;
- React plugin;
- React Hooks plugin;
- JSX a11y plugin;
- Prettier;

## Setup

1. Install the dependencies

```
npm i -D eslint @andyalmeida/eslint-config
```

2. Create a `.eslintrc.json` file extending the config:

```
{
  "extends": "@andyalmeida/eslint-config/react"
}
```
